# perfect_flutter

DevTools-based pixel-perfect overlay for Flutter. Overlay a design image on
top of your running app from the DevTools panel — no widget wrapping, no app
code changes.

See the [repo README](https://github.com/Hankium/perfect_flutter) for usage.

## Install

```yaml
dev_dependencies:
  perfect_flutter: ^0.0.1
```

Then run the app in debug mode and open DevTools — the "Perfect Flutter" tab
will appear.
